Wonderfruit F&B Task Manager
Connect to Google Sheets so the team shares live data. One-time setup per person, ~3 minutes.
1
Create a Google Sheet
Go to sheets.google.com, create a blank spreadsheet named WF F&B Tasks.
2
Open Apps Script
In the sheet click Extensions → Apps Script. Delete all existing code.
3
Paste the Code.gs file
Copy the contents of Code.gs (provided separately) into the editor and press Ctrl+S. Update the email addresses at the top of the file first.
4
Deploy as a web app
Deploy → New deployment → Type: Web app → Execute as: Me → Access: Anyone → Deploy → Authorize → Copy the URL.
5
Set up email notifications
In the Apps Script editor, select setupTrigger from the function dropdown and click Run. This creates a daily 8am check for deadlines.
6
Paste the URL here
Each team member does this once.
